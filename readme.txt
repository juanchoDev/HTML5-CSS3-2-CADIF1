Requerimientos generales:
Según el proyecto adjuntado (Adjunto_desafio3), realizar las siguientes actividades: Nota: Debe enviar exclusivamente el archivo css resultante. Archivo Adjunto

Requerimientos específicos:
1.- Reescribir el archivo mainrds.css Ajuste el banner de forma tal que ocupe el ancho de la pantalla sea cual sea el dispositivo en el que se vea la pagina. Recuerde que el comportamiento de la pagina debe permanecer igual.

2.- Usar el selector hijo directo.
[Selectro hijo directo: > ]
El combinador > selecciona los elementos que son hijos directos del primer elemento.
Sintaxis: A > B
Ejemplo: La regla ul > li se aplicará a todos los elementos

3.- Usar el selector hermano adyacente.

[Selector hermano adyacente: + ]
El combinador + selecciona hermanos adyacentes. Esto quiere decir que el segundo elemento sigue directamente al primero y ambos comparten el mismo elemento padre.
Sintaxis: A + B
Ejemplo: La regla h2 + p se aplicará a todos los elementos que siguen directamente a un elemento

4.- Usar los selectores de atributos según el caso.
Selecciona elementos basándose en el valor de un determinado atributo.
Sintaxis: [attr] [attr=value] [attr*=value]
Ejemplo: [autoplay] seleccionará todos los elementos que tengan el atributo "autoplay" establecido (a cualquier valor).

5.- Puede usar otros selectores pero debe optimizar la mayor parte del código css por medio de los selectores avanzados y el resultado debe ser el mismo al diseño original. Nota: Debe enviar exclusivamente el archivo css resultante.
